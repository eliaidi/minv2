报活模块(包含，报活、卡控；中修、小辅修、额外修程，都走这个报活卡控流程)
	com.repair.report.action.dictFault:故障字典。
	com.repair.report.action.dictProteam：班组。
	com.repair.report.action.jcPlanDate：机车日计划。
	com.repair.report.action.jtReport：机统报活（所有角色报活，都通过这个包）。
	com.repair.report.action.jtReport.fixUsers：检修工人，卡控签认，查询记录等。
	com.repair.report.action.jtReport.lead：工长，卡控签认，查询记录等。
	com.repair.report.action.jtReport.commitLead：交车工长，卡控签认，查询记录等。
	com.repair.report.action.jtReport.qc：质检员，卡控签认，查询记录等。
	com.repair.report.action.jtReport.technician：技术员，卡控签认，查询记录等。
	com.repair.report.action.jtReport.acceptance：验收员，卡控签认，查询记录等。
