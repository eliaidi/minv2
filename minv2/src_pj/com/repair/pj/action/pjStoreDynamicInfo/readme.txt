配件动态信息action

1、AddPjStoreDynamicInfoAction.java                             添加配件动态信息
2、DeletePjStoreDynamicInfoAction.java                          删除配件动态信息通过id，命名中缺少后缀ById
3、FindAllPjStoreDynamicInfoAction.java                         查询配件动态信息数据
4、FindPagePjStoreDynamicInfoByDictFirstUnitIdAction.java       通过大部件(专业)ID查询该大部件下配件动态信息，查找页面 
5、FindPjSDInfoListByPssiIdAction.java                          通过配件静态信息ID查询动态配件集合
6、FindPjStoreDynamicInfoByIdAction.java                        根据Id查询配件动态信息数据
7、FindPjStoreDynamicInfoByPjdCodeAction.java                   配件编码的唯一性
8、QueryPjStoreDynamicInfoAction.java                           配件动态信息管理-->专业分类-->查询， 配件动态信息管理-->车型分类-->查询
9、ToMoveStorePositionDialog.java                               查询所有移库页面的需要显示的信息,查询移库记录
10、ToUpdateDynamicInfoDialog.java                              跳转至修改动态配件对话框
11、UpdatePjStoreDynamicInfoAction.java                         通过id修改配件动态信息，缺少后缀ById
